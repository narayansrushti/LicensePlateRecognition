#  License Plate Recognition System

## Overview
Automatic License Plate Recognition System using CNN, OpenCV and Streamlit.

## Features
- Vehicle Image Upload
- License Plate Detection
- Character Segmentation
- CNN OCR Recognition
- Streamlit Dashboard
